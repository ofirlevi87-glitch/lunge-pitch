# LUNGE — Employee Performance OS for Tech Companies

Pitch site for **LUNGE**, a B2B program offering daily 15–20 min in-seat ergonomic sessions for tech-company employees, complemented by the **Chef Lunge** AI nutrition app.

## Live site

Once GitHub Pages finishes building: **https://gadofir.github.io/lunge-pitch/**

## What it pitches

A daily live-instructor stretching session delivered in the open space, designed to prevent the six silent-but-expensive conditions that affect every tech worker:

1. Carpal Tunnel Syndrome
2. Text Neck / Forward Head Posture
3. Rounded Shoulders
4. Lower Back Pain (L1–L5)
5. Venous Insufficiency
6. Digital Eye Strain

Sold as a 6-week pilot with HR dashboard and aggregate participation metrics.

## The companion app

**Chef Lunge** — already live with active users:

- iOS: https://apps.apple.com/es/app/chef-lunge/id6745457803?l=en-GB
- Android: https://play.google.com/store/apps/details?id=com.nutricloud.ChefLunge

The app is positioned as an **optional add-on**, not a contract gate.
